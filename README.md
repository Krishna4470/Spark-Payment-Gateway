# Spark-Payment-Gateway
It is a custom and open source payment gateway solution that works with paytm and bharat pe by connecting their credentials. and it has features like transaction history, custom payment page, etc.

Requirements :
PHP Version :- 8+
MySQL


Steps To Instal :
1. Upload The Files On Your Server.
2. Firstly You Need To Create A database.
3. and run the install.php (https://yourdoamain.com/install.php) and fill the required credentials.
4. and run the update.php, if required.
5. and If any database table and field are missing or showing the error in your website, open the mySql databse and import "Own-UPI-Gateway.sql' manually in your database.
